Nmap

nmap — свободная утилита, предназначенная для разнообразного настраиваемого сканирования IP-сетей с любым количеством объектов, определения состояния объектов сканируемой сети (портов и соответствующих им служб).

    Самые важные комнды

  • | ifconfig / Узнать свой ip

  • | arp-scan -l / Сканировать локальную сеть

    Сценарий из ролика

  • | apt install nmap / установка

  • | nmap / открыть прогу

  • | nmap scanme.nmap.org / сканируем сайт

  • | nmap 192.168.0.1-254 / Сканирование диапазона

  • | nmap -sA 192.168.0.12 / TCP ACK сканирование ( -sA)

  • | nmap -sP 192.168.0.0/24 / открыть прогу

  • | nmap -PN 192.168.0.12 / открыть прогу

  • | nmap --reason scanme.nmap.org / открыть прогу

    • | rimap --open scanme.nmap.org / открыть прогу

    | nmap -packet-trace scanme.nmap.org / открыть прогу

    | nmap --iflist / открыть прогу

    | nmap / открыть прогу

    | nmap -O --osscan-guess 192.168.0.1/ открыть прогу
